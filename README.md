# Magic Terminal

Magic Terminal brings fully-featured native-like terminals directly to your VS Code sidebar. 

## Features

- **Sidebar Integration**: Access your terminal from the activity bar or sidebar without taking up screen real estate at the bottom.
- **Multiple Terminal Instances**: Easily spawn and switch between multiple terminal sessions.
- **Commands**:
  - `Magic Terminal: Toggle Terminal` - Toggle the Magic Terminal sidebar panel visibility.
  - `Magic Terminal: New Terminal` - Create a new terminal tab in the sidebar.
  - `Magic Terminal: Kill Active Terminal` - Terminate the active sidebar terminal.

## Requirements

No external requirements. The extension uses internal terminal libraries to render terminal windows in the sidebar.

## Usage

Click the terminal icon in the activity bar to open the Magic Terminal view, or use the command palette to execute any of the Magic Terminal commands.
