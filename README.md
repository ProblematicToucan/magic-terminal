# SideTerm

SideTerm brings fully-featured native-like terminals directly to your VS Code sidebar. 

## Features

- **Sidebar Integration**: Access your terminal from the activity bar or sidebar without taking up screen real estate at the bottom.
- **Multiple Terminal Instances**: Easily spawn and switch between multiple terminal sessions.
- **Commands**:
  - `SideTerm: Toggle Terminal` - Toggle the SideTerm sidebar panel visibility.
  - `SideTerm: New Terminal` - Create a new terminal tab in the sidebar.
  - `SideTerm: Kill Active Terminal` - Terminate the active sidebar terminal.

## Requirements

No external requirements. The extension uses internal terminal libraries to render terminal windows in the sidebar.

## Usage

Click the terminal icon in the activity bar to open the SideTerm view, or use the command palette to execute any of the SideTerm commands.
